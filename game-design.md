# 冒险者酒馆 (Adventurer's Tavern)
## 模拟经营 + 肉鸽 + 建造 + 角色扮演

### 游戏概述
玩家经营一家酒馆，每次游戏随机生成酒馆布局、客人、事件。通过建造/升级设施、招募员工、管理资源来发展酒馆，抵御各种随机事件挑战。

### 美术风格
- 简朴像素风 (16x16 或 32x32 像素)
- 温暖色调的木质结构
- 清晰的 UI 图标

### 核心玩法循环
1. **迎接客人** - 随机客人来访，有不同需求
2. **服务客人** - 分配员工满足客人需求
3. **赚取金币** - 服务越好，小费越多
4. **建造升级** - 用金币建造新设施或升级现有设施
5. **随机事件** - 每回合可能出现好事或坏事
6. **肉鸽选择** - 每阶段提供随机选项（如随机设施、随机事件、随机客人）

### 核心系统
- **资源系统**: 金币、声誉、食材、燃料
- **设施系统**: 餐桌、酒桶、壁炉、厨房、客房等
- **员工系统**: 厨师、服务员、调酒师
- **客人系统**: 不同类型客人有不同需求
- **事件系统**: 随机好事/坏事
- **Roguelike 元素**: 随机选择、永久死亡（失败重来）
- **卡牌系统**: 每日结算时提供3张随机卡牌选择

### 卡牌系统详情
每张卡牌拥有稀有度等级:
- **Common (普通)**: 绿色边框，提供基础资源增益
- **Rare (稀有)**: 蓝色边框，提供特殊事件或资源
- **Epic (史诗)**: 紫色边框，提供设施强化
- **Legendary (传奇)**: 橙色边框，提供强力特殊效果

卡牌效果类型:
- `resource_boost`: 直接资源增益 (金币/食材/燃料/声誉)
- `facility_boost`: 设施产出强化
- `special`: 触发特殊事件

### 视觉规格
- 像素比: 16x16 或 32x32
- 缩放: 2x 或 3x 像素完美渲染
- 相机: 2D 俯视图，可缩放

### 技术架构
- 引擎: Godot 4.6.1
- 脚本: GDScript
- 场景结构: 基于 Node2D 的 2D 游戏

### 开发阶段
- **Phase 1**: 项目初始化 + 基础框架
- **Phase 2**: 核心玩法原型
- **Phase 3**: 完整系统实现
